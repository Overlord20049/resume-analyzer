# Resume Analyzer (Python)

A Python-based tool that analyzes resumes (PDF) and identifies missing skills.

## Features
- Extracts text from PDF resumes
- Detects technical skills using keyword matching
- Calculates a resume score
- Suggests missing skills

## Tech Stack
- Python
- PyPDF2

## How to Run
1. Add your resume as `resume.pdf`
2. Run:
   python3 main.py

## Sample Output
- Skills Found
- Skills Missing
- Resume Score
